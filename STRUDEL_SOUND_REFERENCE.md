# Strudel Sound Selection Reference\n\n## What Strudel Sound Types Are Available?\n\n### 1. Synthesizer Waveforms (Procedural)\nThese generate sound mathematically in real-time:\n\n```javascript\ns(\"note\").sound(\"sine\")      // Pure, smooth tone\ns(\"note\").sound(\"triangle\")  // Softer, fewer harmonics\ns(\"note\").sound(\"square\")    // Harsh, digital, bright\ns(\"note\").sound(\"sawtooth\")  // Aggressive, many harmonics\n```\n\n**Metal Use Cases**:\n- **sine**: Bass, sub-frequencies, clean foundation\n- **triangle**: Supporting guitar layers, harmonic blending\n- **square**: Lead guitar, cutting tones, presence\n- **sawtooth**: Heavy rhythms, power chords, aggression\n\n### 2. Noise Types (Procedural)\nThese generate random noise:\n\n```javascript\ns(\"hh\").sound(\"white\")      // Harsh, constant noise\ns(\"hh\").sound(\"pink\")       // Warmer, reduced highs\ns(\"hh\").sound(\"brown\")      // Softest, low-frequency focused\ns(\"perc\").sound(\"crackle\")   // Subtle noise crackles\n```\n\n**Metal Use Cases**:\n- **white/pink**: Hi-hat texture alternatives\n- **brown**: Ambient reverb tails, subtle texture\n- **crackle**: Vintage character, texture\n\n### 3. FM Synthesis (Frequency Modulation)\nModulates oscillators for complex timbres:\n\n```javascript\ns(\"note\").sound(\"sawtooth\").fm(4)      // FM brightness control\ns(\"note\").sound(\"sawtooth\").fmh(1.5)   // FM harmonicity (1=natural, 1.5+=metallic)\n```\n\n**Metal Use Cases**:\n- FM adds metallic character to synths\n- Perfect for extreme metal tones\n\n### 4. Wavetable Synthesis (Sampled Oscillators)\nUses pre-recorded waveforms:\n\n```javascript\ns(\"note\").sound(\"wt_flute\")    // Wavetable flute\ns(\"note\").sound(\"wt_brass\")    // Wavetable brass\n// 1000+ wavetables available from AKWF library\n```\n\n**Metal Use Cases**:\n- Specialized metal wavetables for synth metal\n- More authentic than basic waveforms\n\n### 5. Drum Samples (Pre-recorded)\nCome from classic drum machines and samplers:\n\n```javascript\n// Default drum sounds (any bank):\ns(\"bd\")    // Bass drum / Kick\ns(\"sd\")    // Snare drum\ns(\"hh\")    // Closed hi-hat\ns(\"oh\")    // Open hi-hat\ns(\"cr\")    // Crash\ns(\"rd\")    // Ride\ns(\"ht\")    // High tom\ns(\"mt\")    // Mid tom\ns(\"lt\")    // Low tom\n\n// Available drum machine banks:\n.bank(\"RolandTR909\")   // 90s metal standard (sharp, punchy)\n.bank(\"RolandTR808\")   // 80s classic (warm, iconic)\n.bank(\"RolandTR606\")   // Compact (tight, precise)\n.bank(\"AlesisSR16\")    // Modern sampler (versatile)\n// ... many more banks available\n```\n\n**Metal Recommendation**: **RolandTR909** for authentic metal sound\n\n### 6. Instrument Samples (VCSL Library)\nPre-recorded instrument samples loaded by default:\n\n```javascript\ns(\"gtr\")      // Guitar (acoustic/electric)\ns(\"bass\")     // Bass guitar\ns(\"piano\")    // Piano\ns(\"moog\")     // Synthesizer\ns(\"rhodes\")   // Electric piano\n// 100+ other instruments available\n```\n\n**Metal Use Cases**:\n- Could load distorted guitar samples if available\n- Bass samples for more realistic tone\n- Generally less suitable for algorithmic generation\n\n### 7. Custom Samples\nLoad your own audio files:\n\n```javascript\nsamples({\n  'heavy_gtr': 'https://example.com/distorted-guitar.wav',\n  'metal_bass': 'https://example.com/metal-bass.wav'\n})\n\ns(\"heavy_gtr\")    // Uses your custom sample\n```\n\n**Metal Use Cases**:\n- Professional distorted guitar samples\n- Authentic metal amp recordings\n- Custom drum samples from session\n\n---\n\n## Why We Chose What We Chose\n\n### Guitars: Sawtooth + Triangle + Square\n\n**Sawtooth** = Rich harmonics\n- Contains many overtones (1×, 3×, 5×, 7× fundamental frequency)\n- Creates \"thick\" sound with inherent harmonic complexity\n- When heavily filtered (lpf=300Hz), sounds like distorted guitar\n\n**Triangle** = Softer layering\n- Fewer high-frequency harmonics than sawtooth\n- Provides warmth when combined with sawtooth\n- Reduces harshness while maintaining thickness\n\n**Square** = Cutting lead\n- Contains odd harmonics (1×, 3×, 5×, 7×...)\n- Very digital, bright, hollow-sounding\n- Cuts through mix for lead presence\n- Harsh but less aggressive than sawtooth\n\n### Bass: Sine\n\n**Sine** = Pure, clean foundation\n- Only the fundamental frequency, no harmonics\n- Doesn't interfere with other instruments\n- When compressed heavily (postgain=1.8), creates punchy attack\n- Perfect for sub-bass (frequencies below 50Hz)\n\n### Drums: RolandTR909\n\n**RolandTR909** = Metal standard\n- Iconic in heavy metal, thrash, industrial\n- Kick: Strong attack, punchy character\n- Snare: Metallic crack, no warmth\n- Hi-hat: Tight, precise, digital\n- Instantly recognizable to metal listeners\n\n---\n\n## Sound Selection Decision Tree\n\n```\nWhat do I need?\n│\n├─→ Bass (low frequencies)\n│   └─→ Use: sine\n│       (pure, clean, compresses well)\n│\n├─→ Heavy Power Chords\n│   └─→ Use: sawtooth\n│       (many harmonics = thick tone)\n│\n├─→ Lead/Solo\n│   └─→ Use: square\n│       (cuts through, bright, present)\n│\n├─→ Rhythm Layer\n│   └─→ Use: triangle\n│       (softer than sawtooth, less harsh)\n│\n├─→ Drums (standard)\n│   └─→ Use: RolandTR909\n│       (metal standard, punchy, aggressive)\n│\n├─→ Drums (warmer)\n│   └─→ Use: RolandTR808\n│       (warm, iconic, less aggressive)\n│\n├─→ Drums (sampling era)\n│   └─→ Use: AlesisSR16\n│       (sample-based, modern, versatile)\n│\n└─→ Something unique?\n    └─→ Explore:\n        - Noise types (white/pink/brown)\n        - FM synthesis (add .fm() modulation)\n        - Wavetables (wt_* prefixed sounds)\n        - Custom samples\n```\n\n---\n\n## Processing Creates the Metal\n\nThe sound isn't JUST the waveform—it's the processing:\n\n```javascript\n// Raw sawtooth = harsh but boring\ns(\"note\").sound(\"sawtooth\")\n\n// With metal processing = CRUSHING DISTORTION\ns(\"note\")\n  .sound(\"sawtooth\")\n  .gain(0.8)           // Make it loud\n  .lpf(300)            // Cut high frequencies (distortion effect)\n  .postgain(1.5)       // Crush it hard (compression)\n```\n\n**The secret**: Metal sound = Aggressive waveform + Heavy filtering + High gain\n\n---\n\n## Finding Sounds in Strudel\n\n1. **Go to**: https://strudel.cc/\n2. **Click**: \"sounds\" tab (bottom right)\n3. **Browse**:\n   - `drum-machines` - All drum banks\n   - `instruments` - VCSL library\n   - `user` - Any custom sounds you've loaded\n\n---\n\n## Quick Reference\n\n| Purpose | Sound | Why | Alternative |\n|---------|-------|-----|-------------|\n| Distorted Guitar | sawtooth | Rich harmonics | triangle (softer) |\n| Lead Guitar | square | Cutting, bright | sawtooth |\n| Bass | sine | Pure, clean | triangle |\n| Chords | sawtooth | Thick harmonics | square (brighter) |\n| Kick | RolandTR909 bd | Metal standard | RolandTR808 bd |\n| Snare | RolandTR909 sd | Metallic crack | AlesisSR16 sd |\n| Hi-hat | RolandTR909 hh | Tight, digital | RolandTR808 hh |\n| Texture | pink noise | Warm hi-hats | white noise (harsh) |\n\n---\n\n## Customization Ideas\n\n### Want Heavier Sound?\n- Change `triangle` to `sawtooth` for rhythm guitar 2\n- Use `RolandTR808` for warmer, fatter drums\n- Increase postgain values by 0.2\n\n### Want Cleaner Sound?\n- Change `sawtooth` to `triangle` for rhythm guitars\n- Use `AlesisSR16` for more sampled character\n- Decrease postgain values by 0.3\n- Increase lpf values by 50Hz\n\n### Want More Presence?\n- Use `square` for more guitar tracks\n- Increase all gain values by 0.05-0.1\n- Decrease room values (less reverb = tighter)\n\n### Want Unique Metal Style?\n- Experiment with FM synthesis: `.fm(8).fmh(2.5)`\n- Try `white noise` for extreme hi-hats\n- Use `crackle` sound for vintage character\n- Create custom distorted guitar sample library\n\n---\n\n## The Bottom Line\n\n✅ **Synthesis** (sawtooth/square/sine) = Best for algorithmic generation  \n✅ **TR909 Drums** = Metal standard, no debate  \n✅ **Heavy Processing** (lpf + postgain) = Creates distortion  \n✅ **Layering** (multiple waveforms) = Creates thickness  \n✅ **Tight Sound** (low room) = Metal aesthetic  \n\n**Result**: Authentic metal sound without needing actual guitar recordings!\n"
